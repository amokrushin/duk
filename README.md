## Building from sources

### Install module dependensies, including submodiles `/services` and `/shared`

```bash
yarn
```

### Create `.env` file at the module's root path

### Build docker containers

```bash
npm run build
```

### Start service

```bash
npm start
```

### Stop service

```bash
npm stop
```
